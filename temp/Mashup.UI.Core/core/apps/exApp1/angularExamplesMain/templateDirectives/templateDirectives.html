Basic example of ng-repeat

We used ng-repeat to build a basic table with no style.

Action Done
{{item.action}} {{item.done}} 
    <table class="table">
        <thead>
            <tr>
                <th>Action</th>
                <th>Done</th>
            </tr>
        </thead>
        <tbody>
            <tr ng-repeat="item in vm.items">
                <td>{{item.action}}</td>
                <td>{{item.done}}</td>
            </tr>
        </tbody>
    </table>
ng-repeat="item in vm.items"
NOTE: remove the class="table" statement. You'll see how much bootstrapper does for your view with little effort.

Enumerate Properties

You can use ng-repeat to enumerate all the properties of an object.

Action Done
{{prop}} 
    <table class="table table-bordered table-striped table-hover table-responsive ">
        <thead>
            <tr>
                <th>Action</th>
                <th>Done</th>
            </tr>
        </thead>
        <tbody>
            <tr ng-repeat="item in vm.items">
                <td ng-repeat="prop in item">{{prop}}</td>
            </tr>
        </tbody>
    </table>
Notice that only Action and Done have headings. That beause the rest weren't defined.
You can find a more complete example of this where the header is also dynamically created in the Web Api examples.

ng-repeat /w key

Method of using ng-repeat that returns a key /w each property or object.

{{key}} = {{value}} 
    <tr ng-repeat="(key, value) in vm.items">
        <td>
            {{key}} = {{value}}
        </td>
    </tr>
ng-repeat="(key, value) in items"
The first parameter is always the key and the second is the value.
For object collections the key is always the position.
For properties the key is always the property name.

$index

Returns the position of the current object or property

Position Action Done
{{$index + 1}} {{item.action}} {{item.done}} 
    <table class="table table-bordered table-striped table-hover table-responsive ">
        <thead>
            <tr>
                <th>Position</th>
                <th>Action</th>
                <th>Done</th>
            </tr>
        </thead>
        <tbody>
            <tr ng-repeat="item in vm.items">
                <td>{{$index + 1}}</td>
                <td>{{item.action}}</td>
                <td>{{item.done}}</td>
            </tr>
        </tbody>
    </table>
<td>{{$index + 1}}</td>
The position is zero based but we add one so the position makes more sense to the viewer.

$odd, $even, ng-class

This example uses $odd and $even as variables to determine which class to apply via ng-class.

Action Done
{{item.action}} {{item.done}} 
    <style>
        .odd { background-color: green}
        .even { background-color: blue}
    </style>
                
    <tr ng-repeat="item in vm.items" ng-class="$odd ? 'odd' : 'even'">
        <td>{{item.action}}</td>
        <td>{{item.done}}</td>
    </tr>
ng-class="$odd ? 'odd' : 'even'"
Normally we would use the "table-striped" css class from Twitter Bootstrap to alternate row colors. This was just for demonstration purposes.

ng-if

ng-if removes elements when it's evaluation is false.

Action Done
{{item.action}} {{item.done}} 
    <table class="table table-bordered table-striped table-hover table-responsive ">
        <thead>
            <tr>
                <th>Action</th>
                <th>Done</th>
            </tr>
        </thead>
        <tbody>
            <tr ng-repeat="item in vm.items">
                <td>{{item.action}}</td>
                <td><span ng-if="$first || $last">{{item.done}}</span></td>
            </tr>
        </tbody>
    </table>
<span ng-if="$first || $last">{{item.done}}</span>
If the item is the first or last element then the "Done" column is displayed otherwise ng-if removes it.

ng-repeat-start, ng-repeat-end

Determines the start and end of a repeatign section.

This is item {{$index}}
The action is: {{item.action}}
Item {{$index}} is {{item.done? '' : "not "}} complete 
    <table class="table table-bordered table-striped table-hover table-responsive ">
    <tbody>
        <tr ng-repeat-start="item in vm.items">
            <td>This is item {{$index}}</td>
        </tr>
        <tr>
            <td>The action is: {{item.action}}</td>
        </tr>
        <tr ng-repeat-end>
            <td>Item {{$index}} is {{item.done? '' : "not "}} complete</td>
        </tr>
    </tbody>
    </table>
