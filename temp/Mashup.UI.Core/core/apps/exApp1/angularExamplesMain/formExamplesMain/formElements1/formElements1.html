Basic Validation

This is a simple example of "required" validation

Please enter a name
Please enter an address 
    <form name="shippingForm" novalidate>
            <div class="form-group">
                <label>Name</label>
                <input name="name" class="form-control vp-textbox-input" ng-model="data.shipping.name" required />
                <span class="error" ng-show="shippingForm.name.$error.required">Please enter a name</span>
            </div>
            <div class="form-group">
                <label>Address</label>
                <input name="address" class="form-control vp-textbox-input" ng-model="data.shipping.address" required />
                <span class="error" ng-show="shippingForm.address.$error.required">Please enter an address</span>
            </div>
            <div class="">
                <button ng-click="completeShipping()" ng-disabled="shippingForm.$invalid" class="btn btn-success">Complete Order</button>
            </div>
        </form>
The form element supports 3 purposes
1. Enables validation. AngularJS repurposes the form element for validation.
2. Disable validation the browser might want to perform.
3. Define a variable validation can be referenced with. The name attribute is used.
Without the "novalidate" attribute there might be conflicts between the browser and AngularJS validation.
All controls inside a FORM element have 1 of 2 classes applied to them. ng-invalid or ng-valid. These can be added to the main.css so they apply consistently throughout the application.
The span tags connect to the control via the form name attribute then apply an Angular validation method.
IE: < class="error" ng-show="shippingForm.address.$error.required">Please enter an address</span>
The button is disabled when the ng-disabled directive is added to the button and the form has an invalide control. ng-disabled="shippingForm.$invalid"

Radio controls

Simple radio control example

    <div class="radio" ng-repeat="button in ['None', 'Table', 'List']">
        <label>
            <input type="radio" ng-model="data.mode" value="{{button}}" ng-checked="$first" />
            {{button}}
        </label>
    </div>
Before adding vm.data = {}; to the controller the radio buttons would select but not deselect. Binding the buttons to data.mode didn't work until the data property was created.
