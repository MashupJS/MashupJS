Alerts

Examples of providing user feedback.

This is a very basic implementation. A custom directive is added to a div on the index.html page and bound to the root $scope. Calling alertService.add with 2 parameters defines the type of message you want to display and updates the model bound to the alert div.

    vm.addAlert = function (messageType) {
        alertService.add(messageType, '<h4>Success!</h4> This is a success message!');
        // options
        // alert-success
        // alert-warning
        // alert-info
        // alert-danger
    };
