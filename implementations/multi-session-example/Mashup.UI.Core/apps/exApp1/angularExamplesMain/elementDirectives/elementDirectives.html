ng-show, ng-hide

ng-show, ng-hide, two-way binding

Hidding DOM elements isn't the same as removing them.

ng-show and ng-hide do not add or remove elements from the DOM but ng-if does.

Notice the Done column. The true/false values are replaced with (Incomplete)/(Done)

While we show Incomplete or Done, both spans are created.

# Action Done Email
{{$index + 1}} {{item.action}} (Incomplete) (Done) {{item.contact}} 
    <tr ng-repeat="item in vm.items">
        <td>{{$index + 1}}</td>
        <td>{{item.action}}</td>
        <!--<td>{{item.done}}</td>-->
        <td>
            <span ng-hide="item.done">(Incomplete)</span>
            <span ng-show="item.done">(Done)</span>
        </td>
        <td>{{item.contact}}</td>
        <!--<td ng-repeat="prop in item">{{prop}}</td>-->
    </tr>
<span ng-hide="item.done">(Incomplete)</span>
<span ng-show="item.done">(Done)</span>

ng-if

This example does the same thing as the previous except one span tag is created and one is removed.

# Action Done Email
{{$index + 1}} {{item.action}} (Incomplete) (Done) {{item.contact}} 
    <tr ng-repeat="item in vm.items">
        <td>{{$index + 1}}</td>
        <td>{{item.action}}</td>
        <td>
            <span ng-if="!item.done">(Incomplete)</span>
            <span ng-if="item.done">(Done)</span>
        </td>
        <td>{{item.contact}}</td>
    </tr>
<span ng-if="!item.done">(Incomplete)</span>
<span ng-if="item.done">(Done)</span>
